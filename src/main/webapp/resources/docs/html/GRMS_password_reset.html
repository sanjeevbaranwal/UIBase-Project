GRMS Password reset procedure:



NOTE: Make sure you are using correct credentials to login. It is a common mistake that users attempt to access GRMS with their GEMS credentials!

1. Go to the login page of GRMS.

2. Click on the relevant link: “Forgot your password?”

3. Follow the instructions on the screen.

4. You will have to fill in the User Name.

5. Submit the details.

6. An Access Code will be received on your e-mail address.

7. Copy the access code and click on the link.

8. The section ask for details. Enter the TALEO user name and paste the access code into the prompt.

9. Enter the access code.

10. Create a new password.
