The syntax of this command is:

- NET USE [devicename | *] [\\computername\sharename[\volume] [password | *]] [/USER:[domainname\]username] [[/DELETE] | [/PERSISTENT:{YES | NO}]]

- NET USE [devicename | *] [password | *]] [/HOME]

- NET USE [/PERSISTENT:{YES | NO}]

E.g.: You can use it like this:

net use: H: \\ARLES01\SYS

or

net use H: \\ARLES01\SYS /PERSISTENT:YES

or

net use H: \\ARLES01\SYS /PERSISTENT:N
